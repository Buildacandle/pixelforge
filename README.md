# PixelForge

Bold & colorful agency portfolio website built with Next.js 16, TypeScript, and Tailwind CSS 4.

## Features

- **Neo-brutalist design** with thick borders, hard offset shadows, and vibrant color blocks
- **Hero section** with bold CTA and animated elements
- **Services showcase** for Web Development, E-Commerce, and Mobile Apps
- **3-tier pricing chart** (Starter $300-$1K, Business Pro $1.25K-$5K, App Builder $250-$10K)
- **Monthly support add-ons** (Basic $49/mo, Growth $149/mo, Priority $299/mo)
- **Portfolio grid** with category filtering and placeholder projects
- **Testimonials** with client reviews
- **Contact form** with service selector
- **Fully responsive** (mobile, tablet, desktop)

## Tech Stack

- **Next.js 16** with App Router
- **TypeScript**
- **Tailwind CSS 4**
- **Lucide React** icons
- **Syne + Outfit** fonts (Google Fonts)

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- **Colors**: Edit CSS variables in `app/globals.css`
- **Pricing**: Update tiers in the `Pricing` component in `app/page.tsx`
- **Portfolio**: Replace placeholder projects in the `Portfolio` component
- **Contact info**: Update email, phone, location in the `Contact` component
